Double Machine Learning for Causal Inference under Shared-State Interference

Efficient treatment effect estimation when units interact through markets,
recommender systems, prices, inventory, or other shared states.

Chris Hays  &  Manish Raghavan

Massachusetts Institute of Technology

April 15, 2025  ·  arXiv:2504.08836v1

Today I will present a paper by Chris Hays and Manish Raghavan from MIT. The paper studies causal inference when one unit's treatment, behavior, or outcome can affect later units through a shared state. The key contribution is a general DML framework that still gives asymptotically valid inference under this kind of interference.

Motivation

Standard causal inference assumes no interference between units — this breaks in many real platforms:

🎯
Recommender Systems
One user's engagement changes recommendations shown to future users
Shared state: popularity scores, embeddings, watch history
🛒
Marketplaces
Discounts alter demand, prices, wait times, and inventory
Shared state: inventory level, current price, demand signal
🚗
Ride-Sharing
Treating some riders changes driver availability for others
Shared state: driver pool size, surge pricing, wait times
Even randomized experiments can be biased when a treated unit changes a shared market condition affecting future units. Hover over each card to see the shared state.
In many real systems, SUTVA or no-interference assumptions are too strong. This paper focuses on settings where the interference has structure: it is mediated by an observed shared state.

Core Idea: Shared-State Interference

Interference is not arbitrary — it flows through an observed shared state \(H_t\) that summarizes all past spillovers.

\(H_t\) can be inventory levels, prices, wait times, popularity scores, or recommendation outputs — any mechanism that mediates spillover across units over time.
The key modeling move is to represent spillovers through H_t. This state summarizes how previous units affect later ones.

Formal Model

Observed data at time \(t\):

\[ W_t = \{X_t,\, D_t,\, H_t,\, Y_t\} \]
  • \(X_t\): i.i.d. covariates
  • \(D_t\): binary treatment, independent of previous data but may depend on \(X_t\)
  • \(H_t\): observed shared state
  • \(Y_t(D_t, H_t)\): potential outcome depending on treatment and shared state
Key Markov Assumption
\[ P(H_t \in A \mid W_{1:t-1}) = P(H_t \in A \mid W_{t-1}) \]

Conditional on the previous observation, the next shared state is independent of older history. The full sequence \(W_t\) is a homogeneous Markov chain.

Because X_t is i.i.d. and treatment does not depend on past data, the full sequence W_t is itself a homogeneous Markov chain.

Dependence Conditions

The paper assumes one of two dependence structures:

Condition 1 — Geometric Ergodicity with Detailed Balance

The Markov chain converges to a unique stationary distribution. Correlations decay fast enough for asymptotic inference.

Condition 2 — \(m\)-Dependence

Observations more than \(m\) steps apart are independent:

\[ W_{1:t} \perp W_{t+m+1:T} \]

Natural for switchback experiments.

These assumptions replace the i.i.d. assumption in ordinary DML and allow a central limit theorem to hold.
These assumptions are what allow the authors to control covariance between observations over time and prove a central limit theorem.

Why Double Machine Learning?

DML allows flexible nuisance estimation with valid inference.

Nuisance Functions

  • Outcome model \(f\)
  • Propensity score \(m\)
  • Other learned conditional expectations

DML Mechanism

  • Plug-in prediction term
  • Debiasing correction term
  • Neyman orthogonality — first-order nuisance errors cancel
Naive plug-in ML estimators are biased and converge too slowly. DML is first-order insensitive to nuisance estimation error, enabling flexible ML models with \(\sqrt{T}\)-rate inference.
Naive plug-in ML estimators can be biased and converge too slowly. DML solves this by designing the estimator to be first-order insensitive to nuisance estimation error.

Main General Theorem

Theorem 3.3 — Asymptotic Normality

Under:

  • Shared-state dependence assumptions
  • Smoothness and Neyman orthogonality
  • Regularity and nuisance convergence rates

Then:

\[ \sqrt{T}\,\hat{\sigma}^{-1}\bigl(\psi(W_{1:T};\,\hat{\eta}) - \psi^*\bigr) \Rightarrow N(0,1) \]
Extends standard DML asymptotic normality from i.i.d. data to shared-state interference, by replacing i.i.d. concentration and CLT arguments with Markov-chain or \(m\)-dependent arguments.
This is the paper's main theoretical result. The proof follows the Chernozhukov et al. DML argument, but replaces i.i.d. concentration and CLT arguments with Markov-chain or m-dependent arguments.

Variance Estimation

Variance must account for temporal dependence.

The paper provides two consistent variance estimators:

Geometrically Ergodic Markov Chains

Uses a batch-means style estimator that averages over blocks of observations.

\(m\)-Dependent Data

Uses a plug-in variance estimator with covariance terms up to lag \(m\).

Key practical point: Treating shared states as ordinary i.i.d. covariates may give the same point estimator, but the variance estimate is wrong — observations are correlated over time.
This is a major practical point. Treating shared states as ordinary iid covariates may produce the same point estimator in some cases, but the variance estimate can be wrong because observations are correlated over time.
Application 1 — Observational Settings

Average Direct Effect: Structural Model

Structural equations:

\[ Y_t(D_t,H_t)=f^*(D_t,X_t,H_t)+\tilde{Y}_t \]
\[ H_t(W_{t-1})=h^*(W_{t-1})+\tilde{H}_t \]
\[ D_t=m^*(X_t)+\tilde{D}_t \]

Estimand (ADE):

Average Direct Effect
\[ \psi^*=\frac{1}{T}\sum_{t=1}^T \mathbb{E}[Y_t(1,H_t)-Y_t(0,H_t)] \]

Changes only the focal unit's treatment. Other units' treatment assignment distribution remains fixed.

This estimand changes only the focal unit's treatment. Other units' treatment assignment distribution remains fixed. This makes it a direct effect, not a total system-wide effect.

ADE Estimator

Plug-in term plus debiasing correction:

Score Function \(\phi(W_t;\eta)\)
\[ \begin{aligned} \phi(W_t;\eta) &= f(1,X_t,H_t) - f(0,X_t,H_t) \\[4pt] &\quad +\left(\frac{D_t}{m(X_t)}-\frac{1-D_t}{1-m(X_t)}\right)\bigl(Y_t - f(D_t,X_t,H_t)\bigr) \end{aligned} \]

Nuisances: \(\eta = (f, m)\)

This resembles the standard AIPW estimator — except the outcome model includes the shared state \(H_t\). However, inference is not i.i.d. because \(H_t\) introduces serial dependence.
This looks like the usual AIPW estimator, except the outcome model includes the shared state H_t. However, the inference is not i.i.d. because H_t makes observations dependent over time.

ADE Assumptions and Result

Main requirements:

  • Treatment probabilities bounded away from 0 and 1
  • Outcome functions and residuals have bounded moments
  • \(f\) and \(m\) converge in \(L_2\)
  • Product of nuisance errors is \(o_p(T^{-1/2})\)
Theorem 4.3
\[ \sqrt{T}\,\hat{\sigma}^{-1}\bigl(\hat{\psi}_{\text{ADE}} - \psi^*_{\text{ADE}}\bigr) \Rightarrow N(0,1) \]
Standard DML product-rate conditions apply: if both nuisances converge at \(T^{-1/4}\), their product is fast enough. If the propensity score is known, the outcome model can converge more slowly.
The required nuisance rates are the familiar DML product-rate conditions.

Application 2 — Switchback Experiments

Estimating the Global Average Treatment Effect

Switchback experiments assign the same treatment to blocks of sequential units.

Global Average Treatment Effect (GATE)
\[ \psi^* = \frac{1}{T}\sum_{t=1}^T \mathbb{E}\bigl[Y_t(1,H_t(1)) - Y_t(0,H_t(0))\bigr] \]

Compares a world where all units are treated vs. all units are controlled — includes spillover effects.

Unlike ADE, GATE includes spillover effects. Switchback experiments are designed to support this kind of inference when interference is local in time.
Switchback experiments are designed to support this kind of inference when interference is local in time.

Switchback Model and Estimator

Shared state = previous \(m\) treatments & covariates:

\[ H_t = \text{vec}(D_{t-m:t-1},\, X_{t-m:t-1}) \]
Score Function
\[ \begin{aligned} \phi(W_t;\eta) &= f(1,X_t,H_t(1)) - f(0,X_t,H_t(0)) \\[3pt] &\quad + \Bigl(\tfrac{\mathbf{1}\{D_{t-m:t}=\mathbf{1}\}}{\pi^*(\cdot,1)} - \tfrac{\mathbf{1}\{D_{t-m:t}=\mathbf{0}\}}{\pi^*(\cdot,0)}\Bigr)\\[3pt] &\quad \cdot\,(Y_t - f(D_t,X_t,H_t)) \end{aligned} \]

When does the debiasing term fire?

Full time series →

TT TT | CC CC | TT T
✓ Active — pure window (m+1 = 4)
TT TT
All same treatment → outcome reflects all-treated regime → term contributes
✗ Inactive — mixed window
TT CC
Mixed history → can't isolate treatment regime → term skipped
After enough time under a condition, outcomes reflect the all-treatment or all-control regime. The debiasing term only activates on those pure windows.

Switchback Result

Requirements:

  • Probability of \(m+1\) same-treatment observations bounded away from zero
  • Outcome model has bounded moments
  • Outcome nuisance \(f\) converges in \(L_2\)
  • Propensity probabilities are known (experimenter controls randomization)
Theorem 5.3
\[ \sqrt{T}\,\hat{\sigma}^{-1}\bigl(\hat{\psi}_{\text{GATE}} - \psi^*_{\text{GATE}}\bigr) \Rightarrow N(0,1) \]

Since treatment assignment probabilities are known, the only learned nuisance is the outcome model \(f\).

Since treatment assignment probabilities are known, the only learned nuisance is the outcome model. The result gives efficient inference for GATE in switchback experiments using a DML-style regression adjustment.

Proof Strategy

The proof extends standard DML theory to dependent data in four steps:

1
Decompose Estimator Error
Split \(\hat{\psi} - \psi^*\) into: oracle empirical process + nuisance deviation + second-order remainder
↓
2
Apply Neyman Orthogonality
First-order nuisance bias cancels to zero → only oracle term + \(o_p(T^{-1/2})\) remainder survive
↓
3
Apply Dependent-Data CLT
Replace i.i.d. CLT with Markov-chain CLT (geometric ergodicity) or \(m\)-dependent CLT to handle serial covariance of \(\phi(W_t)\)
↓
4
Prove Variance Consistency
Show batch-means estimator (Markov case) or plug-in with lag-\(m\) covariances (\(m\)-dep case) converges to the true long-run variance \(\sigma^2\)
The novelty is not just adding H_t as a covariate. The proof has to handle covariances between phi(W_t) and phi(W_s). The dependence assumptions guarantee those covariances decay or vanish beyond lag m.

Key Takeaways

Main contributions of the paper:

  • Formalizes shared-state interference as a broadly applicable model.
  • Extends DML to dependent observations generated by shared states.
  • Provides asymptotically normal treatment effect estimators.
  • Provides consistent variance estimators that respect serial dependence.
  • Instantiates the framework for ADE (observational) and GATE (switchback).
  • Demonstrates via simulation that naive methods are biased or have invalid CIs.
Core Message

Interference does not make causal inference impossible if the interference has exploitable structure. Shared-state interference is broad enough to describe many platform settings, and DML gives a way to combine flexible ML with valid statistical inference.

The paper's core message is that interference does not make causal inference impossible if the interference has exploitable structure.

Limitations and Future Work

Current Limitations

  • Requires observed shared state \(H_t\)
  • Interference must be fully mediated through \(H_t\)
  • Requires Markovian structure or \(m\)-dependence
  • Main theorem uses auxiliary data for nuisance estimation
  • Does not cover settings where shared state affects treatment assignment

Future Directions

  • Other estimands: indirect effects, time-discounted effects
  • Other interference structures
  • Empirical applications in real markets or information systems
  • Practical sample splitting under dependent data
The framework is powerful but still structured. If the shared state is hidden, high-dimensional, poorly measured, or if treatment assignment depends on the evolving state, additional work is needed.

Final Summary

One-Sentence Summary

When interference flows through an observed shared state, we can recover valid DML-style treatment effect inference by modeling the dependence and estimating variance correctly.

Practical lesson: In platforms and markets, do not simply ignore interference or treat shared states as i.i.d. covariates. If the system has shared-state structure, use estimators and variance formulas that respect the time dependence.
The practical lesson is clear: in platforms and markets, do not simply ignore interference or treat shared states as iid covariates.
